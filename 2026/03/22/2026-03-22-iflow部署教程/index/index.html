iFLOW部署教程

1、安装条件

  • Node.js 22+（用于直接用命令安装）

  • 4GB+ 内存（电脑运行配置最低）

  • 互联网连接(其本身必须联网)

2、安装iFLOW

重启终端：CMD(Windows + r 输入cmd) 或 PowerShell
运行 `npm install -g @iflow-ai/iflow-cli@latest` 来安装 iFlow CLI
运行 `iflow` 来启动 iFlow CLI
