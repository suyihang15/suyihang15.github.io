网页如何打包成打包exe（很基础的）

1、环境准备

安装Node.js和npm

2、验证是否安装好

在cmd中输入node -v和npm -v

看能不能输出版本号可以就装好了

3、下载要的包

npm install -g nativefier

4、验证包的安装

nativefier –version

5、创建exe文件

新建一个文件夹，这里面装的就是打包后的文件后打开文件夹右键打开cmd后输入nativefier ‘网址’

之后找到exe文件就行
