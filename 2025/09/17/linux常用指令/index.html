Linux基本操作指令

Linux是一种强大的操作系统，提供了许多常用的命令行工具，可以帮助我们管理文件、目录、进程、网络和系统配置等。以下是一些常用的Linux命令及其用法：

一、文件和目录操作

命令 功能说明 示例用法
ls 列出当前目录中的文件和子目录 ```bash

基础用法

ls

显示详细信息（权限、大小、时间等）

ls -l

显示隐藏文件（以.开头的文件）

ls -a

| `pwd` | 显示当前工作目录的绝对路径 | ```bash
pwd
``` |
| `cd` | 切换工作目录 | ```bash
# 切换到指定目录
cd /path/to/directory
# 回到上一级目录
cd ..
# 回到当前用户主目录
cd ~
# 回到上一次所在目录
cd -
``` |
| `mkdir` | 创建新目录 | ```bash
# 创建单个目录
mkdir directory_name
# 递归创建多级目录（如a/b/c）
mkdir -p a/b/c
``` |
| `rmdir` | 删除空目录（仅能删空目录） | ```bash
rmdir directory_name
``` |
| `rm` | 删除文件或目录（慎用！） | ```bash
# 删除文件
rm file_name
# 递归删除目录及其所有内容
rm -r directory_name
# 强制删除（无确认提示，高危）
rm -rf directory_name
``` |
| `cp` | 复制文件或目录 | ```bash
# 复制文件到目标位置
cp source_file destination
# 递归复制目录及其内容
cp -r source_directory destination
# 覆盖前提示（避免误操作）
cp -i source_file destination
``` |
| `mv` | 移动/重命名文件/目录 | ```bash
# 重命名文件/目录
mv old_name new_name
# 移动文件到指定目录
mv file_name /path/to/directory
``` |
| `touch` | 创建空文件/更新文件时间戳 | ```bash
# 创建空文件
touch file_name
# 同时创建多个空文件
touch file1 file2 file3
``` |
| `cat` | 查看/拼接文件内容 | ```bash
# 查看单个文件内容
cat file_name
# 拼接多个文件内容并输出
cat file1 file2
# 查看文件并显示行号
cat -n file_name
``` |

## 二、系统管理
| 命令 | 功能说明 | 示例用法 |
|------|----------|----------|
| `ps` | 查看当前运行的进程 | ```bash
# 显示所有进程（最常用）
ps aux
# 按进程树显示
ps -ef
``` |
| `kill` | 终止指定进程 | ```bash
# 终止指定PID的进程（PID为进程号）
kill process_id
# 强制终止进程（无法忽略）
kill -9 process_id
``` |
| `ifconfig/ip` | 查看/配置网络接口 | ```bash
# 传统查看网络信息（部分系统需安装net-tools）
ifconfig
# 新版查看IP地址（推荐）
ip addr show
# 启动/关闭网卡（eth0为网卡名）
ip link set eth0 up/down
``` |
| `ping` | 测试主机连通性 | ```bash
# 测试与百度的连通性（默认无限次）
ping www.baidu.com
# 测试5次后停止
ping -c 5 www.baidu.com
``` |
| `chmod` | 修改文件/目录权限 | ```bash
# 数字权限（755：所有者读/写/执行，其他读/执行）
chmod 755 file_name
# 符号权限（给所有者添加执行权限）
chmod u+x file_name
``` |
| `chown` | 修改文件/目录所有者/所属组 | ```bash
# 修改所有者
chown owner file_name
# 同时修改所有者和所属组
chown owner:group file_name
# 递归修改目录及其内容的所有者
chown -R owner:group directory_name
``` |
| `df/du` | 查看磁盘使用情况 | ```bash
# 查看磁盘分区使用情况（人性化显示）
df -h
# 查看指定目录的磁盘占用（人性化）
du -h directory_name
# 查看目录总大小（仅显示总和）
du -sh directory_name
``` |
| `mount/umount` | 挂载/卸载文件系统 | ```bash
# 挂载分区到指定目录（sdX1为分区名）
mount /dev/sdX1 /mnt
# 卸载挂载的目录（卸载前确保未被使用）
umount /mnt
# 查看已挂载的文件系统
mount
``` |

## 三、网络操作
| 命令 | 功能说明 | 示例用法 |
|------|----------|----------|
| `ssh` | 远程登录其他Linux主机 | ```bash
# 基础远程登录（默认22端口）
ssh username@remote_host
# 指定端口登录（如2222端口）
ssh -p 2222 username@remote_host
``` |
| `scp` | 安全复制文件（跨主机） | ```bash
# 本地文件复制到远程主机
scp local_file remote_user@remote_host:/remote/directory
# 远程文件复制到本地
scp remote_user@remote_host:/remote/file /local/directory
# 递归复制目录
scp -r local_directory remote_user@remote_host:/remote/directory
``` |
| `wget/curl` | 从网络下载文件 | ```bash
# wget下载文件（直接保存到当前目录）
wget https://example.com/file.zip
# curl下载文件（需加-O参数）
curl -O https://example.com/file.zip
# 断点续传（大文件推荐）
wget -c https://example.com/large_file.zip
``` |

---

### 补充说明
1. 所有命令区分大小写，输入时需注意；
2. 带`-r/-R`参数表示**递归**，适用于目录操作；
3. 带`-h`参数表示**人性化显示**（如大小显示为KB/MB，而非字节）；
4. 执行系统级命令（如mount、chown）通常需要加`sudo`获取管理员权限。
